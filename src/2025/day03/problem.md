# Day 3: Lobby

## Problem Description

We are given a list of battery banks (lines of digits).
For each bank:
- Part 1: Select **exactly 2** batteries (digits) to form the largest possible number.
- Part 2: Select **exactly 12** batteries (digits) to form the largest possible number.
The relative order of digits must be preserved.

The final answer is the sum of these maximum numbers across all banks.

## Input

- Multiple lines, each representing a bank of batteries.
- Each line contains a string of digits.

## Output

- Part 1: Sum of max 2-digit numbers.
- Part 2: Sum of max 12-digit numbers.
