# Day 5: Cafeteria

## Problem Description

We are given a list of fresh ingredient ID ranges (e.g., `3-5`, `10-14`).
Part 1 asked to check specific IDs.
Part 2 asks to find the **total count of distinct integers** covered by the union of these ranges.

The second part of the input (list of IDs) is ignored in Part 2.

## Input

- Multiple lines containing ranges `start-end`.
- A blank line.
- (Ignored) Multiple lines containing single integers.

## Output

- Part 1: Count of given IDs that are fresh.
- Part 2: Total number of distinct integers in the union of all ranges.
