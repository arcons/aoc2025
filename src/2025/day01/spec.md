# Spec for Day 1

## Part 1

### Function Signature

`part1(input: string): number`

### Input Format

The input is a multi-line string, where each line represents a rotation instruction.
Each instruction consists of a single character ('L' or 'R') followed by a number.

Example:
```
L68
L30
R48
```

### Output Format

The function should return a single number, which is the count of how many times the dial points to 0 after a rotation is performed.

### Test Cases

#### Example 1

- **Input**:
```
L68
L30
R48
L5
R60
L55
L1
L99
R14
L82
```
- **Expected Output**: `3`
